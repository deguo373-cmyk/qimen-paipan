## 二、吉神详解

---

### 1. 天乙贵人

**【查法】**
以日干为主，见四柱地支：
- 甲戊庚 → 牛羊（丑、未）
- 乙己 → 鼠猴（子、申）
- 丙丁 → 猪鸡（亥、酉）
- 壬癸 → 兔蛇（卯、巳）
- 辛 → 虎马（寅、午）

口诀：**甲戊庚牛羊，乙己鼠猴乡，丙丁猪鸡位，壬癸兔蛇藏，辛逢虎马上。**

**【属性】** 大吉

**【含义与解读】**
天乙贵人是八字中最重要的吉神之首，主贵人相助、逢凶化吉、得人提携。

- **年柱见**：祖上或长辈有贵人，出身环境较好，少年运顺利。
- **月柱见**：得父母、兄弟、同事之助，青年时期人脉好。
- **日柱见**：自身自带贵气，夫妻和睦，一生多得人助，日坐贵人（丁亥、丁酉、癸巳、癸卯）最殊胜。
- **时柱见**：晚运亨通，下属得力，子女有出息。

**组合解读：**
- 天乙贵人 + 天德/月德：贵人中有德者，辅助力极大，是非不侵。
- 天乙贵人 + 文昌：贵人多为文化界、学术圈人士。
- 天乙贵人 + 桃花：贵人中多异性助力，但也易有感情纠葛。
- 天乙贵人逢空亡：贵人力量减弱，空有其名，关键时刻帮不上忙。
- 贵人合克日主：名为贵人，实则约束，须防以"帮助"之名行控制之实。

**应用场景：** 看一个人是否一生有人相助、遇事有无解围之人。八字无天乙贵人者，凡事多靠自身努力，贵人缘浅。

### 2. 天德

**【查法】**
以月令为主，见四柱地支或天干：
- 正月（寅月）见丁
- 二月（卯月）见申（坤）
- 三月（辰月）见壬
- 四月（巳月）见辛
- 五月（午月）见亥（乾）
- 六月（未月）见甲
- 七月（申月）见癸
- 八月（酉月）见寅（艮）
- 九月（戌月）见丙
- 十月（亥月）见乙
- 十一月（子月）见巳（巽）
- 十二月（丑月）见庚

口诀：**正丁二申三壬四辛，五亥六甲七癸八寅，九丙十乙子巳丑庚。**

**【属性】** 大吉

**【含义与解读】**
天德为天道之德，主心地善良、品行端正、逢凶化吉。天德入命者，即使格局不高，亦能得祖上荫庇，一生少灾少难。

- **年柱见**：祖上有德，家风纯良。
- **月柱见**：得父母荫庇，性情温和厚道。
- **日柱见**：自身修养好，待人宽厚，婚姻吉祥。
- **时柱见**：晚年安乐，子孙贤孝。

**组合解读：**
- 天德 + 月德（"天月二德"）：最大吉组合，鬼神不侵，刑狱不临，大病不死。
- 天德 + 天乙贵人：德高望重之人，众人敬服。
- 天德逢冲刑：虽有德心，但被人误解或道德绑架。

**应用场景：** 刑事案件、重大疾病、官非诉讼中，天德是关键的救应神——有之则大事化小。

### 3. 月德

**【查法】**
以月令为主，见四柱天干：
- 寅午戌月 → 丙
- 亥卯未月 → 甲
- 申子辰月 → 壬
- 巳酉丑月 → 庚

口诀：**寅午戌月丙，亥卯未月甲，申子辰月壬，巳酉丑月庚。**

**【属性】** 大吉

**【含义与解读】**
月德为月令之德，主人仁慈、温和、有福气。与天德并称"天月二德"，是命理中最为尊贵的两种吉神。

- **年柱见**：祖上有善行，门风好，自幼受良好家教。
- **月柱见**：心地仁慈，性格温良，贵人运强。
- **日柱见**：配偶贤良，婚姻美满。
- **时柱见**：晚年有福，子女善良。

**组合解读：**
- 月德 + 天德（天月二德同临）：一生福厚，百事顺遂，多寿多福。
- 月德 + 食神：心地特别善良，有宗教缘。
- 月德 + 羊刃：以德化煞，虽刚不暴，反成威仪。

**应用场景：** 看一个人是否有"福德"——即天然的福气和化解危机的能力。天月二德皆无者，一生需更谨慎。

### 4. 天赦

**【查法】**
- 春季（寅卯辰月）：戊寅
- 夏季（巳午未月）：甲午
- 秋季（申酉戌月）：戊申
- 冬季（亥子丑月）：甲子

口诀：**春戊寅，夏甲午，秋戊申，冬甲子。**

天赦为日柱或四柱中见上述干支。以日柱见之为最有力。

**【属性】** 大吉

**【含义与解读】**
天赦是上天赦免罪过的神煞，主逢凶化吉、遇难成祥，特别是涉及官非诉讼、法律纠纷时有化解之力。

- **日柱为天赦**：一生少犯大错，即使偶尔失误也能被谅解或补救。
- **年月天赦**：家庭环境宽松，父母开明。
- **时柱天赦**：晚年通达，能得善终。

**组合解读：**
- 天赦 + 天月德：真正的大赦格局，一生无牢狱之灾。
- 天赦遭刑冲：赦免之力打折，虽能脱罪但需付出代价。
- 天赦遇空亡：虚有其名，关键时无用。

**应用场景：** 极其适合看官非诉讼——命中带天赦者，即使犯法也容易被宽大处理。

### 5. 福星贵人

**【查法】**
以日干为主，见四柱地支：
- 甲 → 寅、子
- 乙 → 卯、丑
- 丙 → 寅、子
- 丁 → 亥、酉
- 戊 → 巳、亥
- 己 → 巳、亥
- 庚 → 巳、亥
- 辛 → 午、戌
- 壬 → 巳、亥
- 癸 → 巳、亥

口诀：**甲丙相邀入虎乡（寅），更游鼠穴最高强；乙丁逢卯未/丑，戊庚巳亥是福星；己壬癸见巳亥，辛到午戌福寿长。**

**【属性】** 吉

**【含义与解读】**
福星贵人主人福泽深厚，一生少忧，不愁吃穿，心胸宽广。福星入命者，即使不够聪明或不够努力，也往往能过上不错的生活。

- **年柱见**：祖业丰裕，出身无忧。
- **月柱见**：青年时期资源多，不缺钱花。
- **日柱见**：自身有福气，配偶也能带来福运。
- **时柱见**：晚年享福，子女成器。

**组合解读：**
- 福星 + 天乙贵人：福气与贵气兼备，富贵双全。
- 福星 + 驿马：动中得福，越忙越好运。
- 福星 + 劫煞/亡神：福气被破，需要后天努力弥补。

**应用场景：** 判断一个人是否"自带福气"——福星贵人重者，命好不费力；无者，福需自求。

### 6. 太极贵人

**【查法】**
以日干为主，见四柱地支：
- 甲、乙 → 子、午
- 丙、丁 → 卯、酉
- 戊、己 → 辰、戌、丑、未
- 庚、辛 → 寅、亥
- 壬、癸 → 巳、申

口诀：**甲乙子午丙丁卯，戊己辰戌丑未寻，庚辛寅亥壬癸巳申，太极贵人神鬼钦。**

**【属性】** 吉

**【含义与解读】**
太极贵人主聪明好学、喜神秘、有灵性，适合玄学、哲学、宗教、艺术等领域。命中带太极贵人者，对阴阳五行、命理风水有天然的领悟力。

- **年柱见**：家学渊源，祖上有从事玄学、道术之人。
- **月柱见**：青少年时期即对哲学、宗教感兴趣。
- **日柱见**：自身悟性高，钻研能力强，日坐太极最有研究精神。
- **时柱见**：晚年倾心修道、修行，或善终有归处。

**组合解读：**
- 太极 + 华盖：标准的"玄学命"，适合深入研究命理术数。
- 太极 + 文昌：学术能力强，能做严谨的学术研究。
- 太极 + 空亡：灵感时有时无，学易容易半途而废。
- 太极 + 羊刃/劫煞：走火入魔倾向，需防执念太深。

**应用场景：** 判断一个人是否适合学命理、玄学。太极贵人旺者，一点即通。

### 7. 文昌贵人

**【查法】**
以日干为主，见四柱地支：
- 甲 → 巳
- 乙 → 午
- 丙 → 申
- 丁 → 酉
- 戊 → 申
- 己 → 酉
- 庚 → 亥
- 辛 → 子
- 壬 → 寅
- 癸 → 卯

口诀：**甲乙巳午丙丁申，戊己申酉庚辛亥，壬癸寅卯是文昌。**

**【属性】** 吉

**【含义与解读】**
文昌贵人主聪明才智、学业优秀、文笔出众。文昌入命者，读书考试运好，逻辑思维清晰，适合文化教育、出版传媒、法律等领域。

- **年柱见**：少时聪明，读书成绩好，得祖上文化熏陶。
- **月柱见**：青少年时期学业鼎盛，考试运佳。
- **日柱见**：文雅博学，有书卷气，配偶也有文化。
- **时柱见**：晚年仍好学不辍，著书立说，传授后学。

**组合解读：**
- 文昌 + 学堂词馆：学运最佳组合，考试独占鳌头。
- 文昌 + 天乙贵人：贵人即为师长、前辈，求学路上多得提携。
- 文昌 + 桃花：才子佳人型，文采风流但有感情干扰。
- 文昌 + 劫煞：为文惹祸，笔下有锋芒，容易得罪人。

**应用场景：** 看考试运、学业前景——文昌不落空亡者，读书至少中上水平。

### 8. 国印贵人

**【查法】**
以日干为主，见四柱地支：
- 甲 → 戌
- 乙 → 亥
- 丙 → 丑
- 丁 → 寅
- 戊 → 丑
- 己 → 寅
- 庚 → 辰

**【属性】** 吉

**【含义与解读】**
国印贵人象征权力、公章、凭信，主掌权、诚实、有威信。适合在政府机关、事业单位、大型企业中担任管理职务。

- **年柱见**：出身官宦或公务员家庭。
- **月柱见**：有组织管理能力，适合在体制内发展。
- **日柱见**：为人诚信，有领导力，能服众。
- **时柱见**：晚年得实权，或退休后仍有社会影响力。

**组合解读：**
- 国印 + 天乙贵人：官贵双全，既有实权又有人脉。
- 国印 + 禄神：俸禄优厚，职位稳固。
- 国印逢冲破：权力被架空，公章被夺之象。
- 国印 + 空亡：有名无实，虚职无权。

**应用场景：** 判断一个人是否有"掌权"的潜质，是否适合走仕途或管理路线。

### 9. 金舆

**【查法】**
以日干为主，见四柱地支：
- 甲 → 辰
- 乙 → 巳
- 丙 → 未
- 丁 → 申
- 戊 → 未
- 己 → 申
- 庚 → 戌
- 辛 → 亥
- 壬 → 丑
- 癸 → 寅

口诀：**甲龙乙蛇丙丁羊，丁猴戊羊己猴乡，庚犬辛猪壬牛位，癸逢虎上是金舆。**

**【属性】** 吉

**【含义与解读】**
金舆为"金车"，主富贵、出行顺利、生活优渥。金舆入命者，容易拥有好车、房产，生活品质高，且享福较容易。

- **年柱见**：家境富足，出行有车马之便。
- **月柱见**：青年时期即能享受较好物质生活。
- **日柱见**：一生生活舒适，配偶条件好。
- **时柱见**：晚年富足，子孙孝顺。

**组合解读：**
- 金舆 + 驿马：出行频繁且顺遂，现代对应为"出差变旅游"。
- 金舆 + 禄神：富且贵，生活品质极高。
- 金舆逢冲破：虽有车房但贷款压力大，或名义上富有实则手紧。
- 金舆 + 空亡：表面光鲜，内里空虚。

**应用场景：** 判断一个人的物质生活质量和享福程度。

### 10. 禄神

**【查法】**
以日干为主，见四柱地支（临官之位）：
- 甲 → 寅
- 乙 → 卯
- 丙 → 巳
- 丁 → 午
- 戊 → 巳
- 己 → 午
- 庚 → 申
- 辛 → 酉
- 壬 → 亥
- 癸 → 子

口诀：**甲寅乙卯丙巳丁午，戊巳己午庚申辛酉，壬亥癸子是禄位。**

个人命中有禄神，称为"食禄"；八字禄神多现，称为"禄多身弱"需结合身强身弱判断。

**【属性】** 吉（但需身强方能任）

**【含义与解读】**
禄神即官禄、食禄，代表薪水、俸禄、福气。禄神入命者，最直接的体现就是工作收入稳定，有固定经济来源。

- **年柱见（岁禄）**：祖上为官或经商，家庭经济基础好。
- **月柱见（建禄格）**：白手起家，靠自己奋斗，青年时期辛苦但有成就。
- **日柱见（坐禄/专禄）**：自尊心强，有主见，不愿受人约束，适合专业技术路线。
- **时柱见（归禄）**：晚年富足，福气归身。

**组合解读：**
- 禄神 + 官星："官禄双全"，为官有俸，名利双收。
- 禄神逢冲（如甲禄寅遇申冲）：破禄、丢工作、降职之象。

- 多位禄神无制：身弱不胜禄，反而为财所累。

**应用场景：** 看工作收入的稳定性和职业层级。禄神是判断一个人"饭碗"硬不硬的关键指标。

### 18. 三奇贵人

**【查法】**
四柱中天干顺排出现特定组合：
- **天上三奇**：甲年/月/日、戊、庚（顺序不能乱）
- **人中三奇**：乙、丙、丁（顺序不能乱）
- **地下三奇**：壬、癸、辛（顺序不能乱）

以年、月、日三柱天干顺排出现为真。也有说法以日、月、时三柱为准。

口诀：**天上三奇甲戊庚，人中三奇乙丙丁，地下三奇壬癸辛。**

**【属性】** 大吉

**【含义与解读】**
三奇贵人是极高层次的吉格。三奇入命者，往往有非凡的才能、特殊的际遇，容易在某个领域成为顶尖人物。三奇主"奇"，即与众不同、出类拔萃。

- **天上三奇（甲戊庚）**：得贵人提拔而显贵，多为政界、商界精英。
- **人中三奇（乙丙丁）**：凭学识才华而闻名，多为文化、艺术、学术界人才。
- **地下三奇（壬癸辛）**：以智慧谋略取胜，多为谋士、策略家。

**组合解读：**
- 三奇 + 天月二德：真正的大格局，德才兼备，流芳百世。
- 三奇 + 华盖：格局清奇，不入凡俗，常出高僧大德或艺术家。
- 三奇在年柱：祖上风光，出身不凡。
- 三奇在时柱：晚年煊赫，青史留名。

**注意：** 三奇贵人须得地支配合有力，无冲破、无空亡，方能显贵。否则只是空有虚名。

**应用场景：** 看一个人是否有"非凡之命"，是否能在某个领域达到顶尖高度。

### 21. 桃花（咸池）

**【查法】**
以年支/日支为主，见四柱地支（三合局之沐浴位）：
- 寅午戌年 → 卯
- 巳酉丑年 → 午
- 申子辰年 → 酉
- 亥卯未年 → 子

口诀：**寅午戌见卯，巳酉丑见午，申子辰见酉，亥卯未见子。**

**【属性】** 中性（吉凶依组合而定）

**【含义与解读】**
桃花主异性缘、感情、艺术、审美、风流韵事。桃花本身为中性神煞——得制化则为"墙内桃花"，主夫妻恩爱、人缘好、有艺术气质；无制化则成"墙外桃花"，主感情泛滥、婚姻不稳、风流债多。

- **年柱见（墙内桃花）**：早年感情单纯，初恋美好，或艺术天赋高。
- **月柱见（墙内桃花）**：正当青春，异性缘佳，容易早恋但能成好事。
- **日柱见（日坐桃花）**：配偶相貌好，自身魅力足，但需防婚姻不稳。
- **时柱见（墙外桃花）**：晚节看似风流，或子女在感情上多波折。

**组合解读：**
- 桃花 + 天乙贵人：贵人多为异性，能将异性缘转化为事业助力。
- 桃花 + 文昌：风流才子/才女，有艺术气质，文艺创作佳。
- 桃花 + 羊刃：情伤、暴烈的情爱关系，易因感情惹祸。
- 桃花 + 劫煞：桃花劫，因感情破财或招灾。
- 桃花 + 空亡：桃花虚浮，感情之事空欢喜。
- 三合局见多位桃花：桃花泛滥，一生情债多。

**实用判断：**
- 桃花在年/月为"早桃花"，易早恋早婚
- 桃花在日/时为"晚桃花"，婚后仍有吸引力
- 桃花带合：感情纠葛，三角关系
- 桃花+沐浴：情欲旺盛

**应用场景：** 看感情运、异性缘、婚姻稳定性。桃花是八字中判断感情之事最重要的神煞。

### 22. 驿马

**【查法】**
以年支/日支为主，见四柱地支（三合局之驿马位）：
- 寅午戌年 → 申
- 巳酉丑年 → 亥
- 申子辰年 → 寅
- 亥卯未年 → 巳

口诀：**寅午戌见申，巳酉丑见亥，申子辰见寅，亥卯未见巳。**

**【属性】** 中性（偏吉，但劳碌）

**【含义与解读】**
驿马主动态、奔波、旅行、变动、迁移、出国。驿马入命者，一生多动少静，适合从事需要出差、外勤、外贸、运输、外交等行业。

- **年柱见**：少年离家，或祖上迁移。
- **月柱见**：青年时期奔波劳碌，但动中得财。
- **日柱见**：身忙心也忙，坐不住，配偶或自身为外地人。
- **时柱见**：晚年仍不得闲，或子女在外地发展。

**组合解读：**
- 驿马 + 禄神：动中求财，越忙越富，是"富马"。
- 驿马 + 财星：经商之命，适合外贸、物流、旅游业。
- 驿马 + 官星：出差多的工作，或异地升官。
- 驿马 + 桃花：异地情缘，或外出时桃花运旺。
- 驿马逢合（"马被绊住"）：想动动不了，外出计划受阻。
- 驿马逢冲：奔波加剧，劳碌之极。
- 多位驿马：终身奔波，居无定所。

**实用判断：**
- 驿马在年柱：早年离开家乡
- 驿马在时柱：晚年出国旅居
- 驿马+空亡：忙而无功，白跑

**应用场景：** 看一个人工作生活方式是动是静，是否适合出远门、移民、从事外贸等。

### 23. 华盖

**【查法】**
以年支/日支为主，见四柱地支（三合局之墓库位）：
- 寅午戌年 → 戌
- 巳酉丑年 → 丑
- 申子辰年 → 辰
- 亥卯未年 → 未

口诀：**寅午戌见戌，巳酉丑见丑，申子辰见辰，亥卯未见未。**

**【属性】** 中性（偏孤，但也主智慧）

**【含义与解读】**
华盖主孤独、清高、才华、宗教缘、艺术天赋。华盖入命者，有与众不同的气质，性格孤傲，不喜社交，内心丰富，常有奇思妙想。

- **年柱见**：少年孤独，不合群，早慧。
- **月柱见**：与父母缘分薄，或过早独立。
- **日柱见**：婚姻孤独感重，即使有配偶也常有孤军奋战之感。
- **时柱见**：晚年倾向归隐、修行，看破红尘。

**组合解读：**
- 华盖 + 太极贵人：标准玄学命格，适合研究命理、风水、佛法。
- 华盖 + 文昌/学堂：文学艺术天才，如李白、曹雪芹。
- 华盖 + 空亡：遁入空门之象，出家人多有此组合。
- 华盖 + 驿马：云游四方，行脚僧/旅行作家。
- 华盖 + 桃花：情路孤独，艺术家的爱情大多凄美。
- 华盖逢冲破：才华被压制，空有抱负无法施展。

**实用判断：**
- 华盖重（2个以上）：特立独行，不适合在普通职场工作
- 华盖+空亡+太极：出尘之客

**应用场景：** 判断一个人的孤独感、才华方向（艺术vs玄学vs学术），以及是否适合修行或自由职业。

### 24. 魁罡

**【查法】**
日柱为以下四日之一：
- 庚辰
- 庚戌
- 壬辰
- 戊戌

口诀：**庚辰庚戌壬辰戊戌，四日魁罡贵无敌。**

**【属性】** 中性（凶则凶极，吉则大贵）

**【含义与解读】**
魁罡是"魁星"与"罡星"的合称，主权威、刚猛、果决、威仪。魁罡入命者，性格刚烈，不甘人下，有领导欲，做事雷厉风行。

**各日柱特点：**
- **庚辰日（魁罡+白虹贯日）**：刚毅正直，有威仪，适合军警司法。
- **庚戌日（魁罡+火金相战）**：急躁刚猛，容易冲动，但仍不失正义感。
- **壬辰日（魁罡+龙归大海）**：智勇双全，有谋略，成大器者多。
- **戊戌日（魁罡+土重金埋）**：聚财能力强，但也容易固执己见。

**组合解读：**
- 魁罡 + 羊刃/劫煞：凶神聚集，有暴力倾向或非正常死亡风险。
- 魁罡 + 天德/月德：以德制暴，化刚为柔，反而成就大事业。
- 魁罡 + 官星：执法者，法官、警察、军人。
- 魁罡逢冲破（辰戌冲最烈）：魁罡被破，性情大变，从极端刚烈到极端消沉。
- 双魁罡以上：性格极度刚硬，一生多争斗。

**重要规则：** 魁罡不喜财官（尤其是正财正官），因魁罡是将帅之命，过于追求钱财反而堕了格局。魁罡格最喜运行身旺之地。

**应用场景：** 判断一个人的性格刚烈程度和是否适合从事军警、管理、竞争性强的工作。

### 25. 金神

**【查法】**
日柱或时柱为以下三日之一：
- 乙丑
- 己巳
- 癸酉

口诀：**乙丑己巳癸酉，金神入火乡为贵。**

**【属性】** 中性（喜火炼，忌水沉）

**【含义与解读】**
金神为"杀伐之神"，主刚强、果决、有魄力。金神入命者，性格刚烈不屈，有闯劲，适合在军队、警察、创业等领域发展。金神格最喜见火——火炼金神方成大器。

**各日柱特点：**
- **乙丑日**：金神带湿土，需要火来烘干方能显贵。
- **己巳日**：金神自带巳火，本就带火气，最容易成器。
- **癸酉日**：金神带水，水冷金寒，需要强火驱寒。

**组合解读：**
- 金神 + 火（尤其是午、巳）："金神入火乡"，富贵之命，极其刚强有成。
- 金神 + 水：水多金沉，反为奴仆之命。
- 金神 + 羊刃：暴烈之极，容易走极端。
- 金神 + 天月二德：以德化刚，成大事者。

**应用场景：** 金神格的成就高度取决于是否有"火"来锻炼，无火则刚猛无措。

### 26. 羊刃

**【查法】**
以日干为主，见四柱地支（帝旺之位）：
- 甲 → 卯
- 乙 → 辰（一说乙见寅）
- 丙 → 午
- 丁 → 未
- 戊 → 午
- 己 → 未
- 庚 → 酉
- 辛 → 戌（一说辛见申）
- 壬 → 子
- 癸 → 丑

口诀：**甲卯乙辰丙午丁未，戊午己未庚酉辛戌，壬子癸丑羊刃位。**

（注：关于乙、辛、丁、己的羊刃，古籍有分歧，以上依《渊海子平》主流说法。）

**【属性】** 凶

**【含义与解读】**
羊刃是十二长生中帝旺之位，代表力量到达极致，刚强过甚。羊刃入命者，性格刚烈、好强、不服输、易与人争斗。用得好则成大将之才，用不好则祸害自身。

- **年柱见**：祖上性格刚强，或出身武职家庭。
- **月柱见**：青年时期好斗，容易与人冲突。
- **日柱见（日刃）**：自身性格刚烈，配偶也强势，婚姻易有摩擦。
- **时柱见**：晚年仍不服老，子女性格刚强。

**组合解读：**
- 羊刃 + 官杀（尤其七杀）：羊刃驾杀，大将军之命，最高成就。
- 羊刃 + 财星：羊刃夺财——容易因赌、投资、合伙破财。
- 羊刃 + 劫煞：凶上加凶，血光之灾风险高。
- 羊刃 + 天月二德：德能制刃，化凶为吉，反成威权。
- 羊刃逢冲（卯酉冲、子午冲）：爆发冲突之年，需慎防。
- 多位羊刃：六亲缘薄，性格极其刚暴。

**应用场景：** 判断一个人是否有暴力倾向、竞争意识有多强，以及在关键年份是否有血光之灾的风险。

### 27. 劫煞

**【查法】**
以年支为主，见四柱地支（三合局之绝位）：
- 寅午戌年 → 亥
- 巳酉丑年 → 寅
- 申子辰年 → 巳
- 亥卯未年 → 申

口诀：**寅午戌见亥，巳酉丑见寅，申子辰见巳，亥卯未见申。**

**【属性】** 凶

**【含义与解读】**
劫煞是劫夺之神，主破财、失窃、被人抢夺、意外损失。劫煞最怕与财星同柱或相会——代表钱财被夺之象。劫煞也主突发性的灾难、意外事故。

- **年柱见**：祖业破败，或家族曾有被劫之灾。
- **月柱见**：青年时期破财，或被朋友连累。
- **日柱见**：自身财物难聚，容易遭小偷或被骗。
- **时柱见**：晚年仍需防破财，或因子女破财。

**组合解读：**
- 劫煞 + 财星（尤其正财）：最直接的"破财组合"，钱难存住。
- 劫煞 + 羊刃：暴力劫夺，社会新闻中常见此组合。
- 劫煞 + 七杀：突发性灾祸，意外事故。
- 劫煞 + 天乙贵人/天德：贵人化解劫难，有惊无险。
- 劫煞逢空亡：劫难落空，反为吉。

**应用场景：** 看破财风险、意外事故。劫煞重的年份需要特别谨慎对待财务和安全。

### 28. 灾煞

**【查法】**
以年支为主，见四柱地支（三合局之胎位）：
- 寅午戌年 → 子
- 巳酉丑年 → 卯
- 申子辰年 → 午
- 亥卯未年 → 酉

口诀：**寅午戌见子，巳酉丑见卯，申子辰见午，亥卯未见酉。**

**【属性】** 凶

**【含义与解读】**
灾煞主灾祸、疾病、外伤、意外。灾煞与劫煞相近，但劫煞偏重"财物被夺"，而灾煞偏重"身体受伤、生病、天灾"。

- **年柱见**：祖上有水火灾难，或家族遗传病。
- **月柱见**：青年时期健康不佳，容易出事故。
- **日柱见**：自身一生多病多灾，需格外注意安全。
- **时柱见**：晚年有健康隐患。

**组合解读：**
- 灾煞 + 羊刃：血光之灾，车祸、手术、外伤。
- 灾煞 + 七杀：暴疾恶病，或突发灾难。
- 灾煞 + 天医：有病有药，虽灾能愈。
- 灾煞 + 天月二德：凶中有救，关键时刻化险为夷。

**应用场景：** 看一个人一生的灾祸倾向和健康风险。

### 29. 勾绞

**【查法】**
以年支为主，见四柱地支：
- 阳男、阴女：年支顺数三位（如子见卯）
- 阴男、阳女：年支逆数三位（如子见酉）

更简单的查法——勾神为年支前三位，绞神为年支后三位：
- 子 → 勾卯、绞酉
- 丑 → 勾辰、绞戌
- 寅 → 勾巳、绞亥
- 卯 → 勾午、绞子
- 辰 → 勾未、绞丑
- 巳 → 勾申、绞寅
- 午 → 勾酉、绞卯
- 未 → 勾戌、绞辰
- 申 → 勾亥、绞巳
- 酉 → 勾子、绞午
- 戌 → 勾丑、绞未
- 亥 → 勾寅、绞申

口诀：**子卯丑辰寅巳卯午，辰未巳申午酉未戌，申亥酉子戌丑亥寅（勾绞相随）。**

**【属性】** 凶

**【含义与解读】**
勾绞主官非、诉讼、纠缠、阻滞。勾神为勾连，绞神为绞杀，二者常并见。勾绞入命者，容易遇到纠缠不清的事情，如合同纠纷、欠款纠纷、口舌官司等。

- **年柱见**：家庭常有纠纷，或祖上有诉讼之事。
- **月柱见**：青年时期容易与人发生矛盾，官司口舌多。
- **日柱见**：自身易被纠缠，做事诸般不顺。
- **时柱见**：晚年仍有官非之扰。

**组合解读：**
- 勾绞 + 劫煞：因财起争讼，被劫后打官司。
- 勾绞 + 官非星（如亡神）：容易惹上官司。
- 勾绞 + 天乙贵人：虽有纠纷但有贵人相助解决。
- 勾绞逢冲：矛盾激化，官司升级。

**应用场景：** 看一个人一生中是否容易遇到官非诉讼和纠纷问题。

### 30. 元辰（大耗）

**【查法】**
以年支为主，见四柱地支：
- 阳男、阴女：年支对冲再顺数一位（如子对冲为午，顺数为未）
- 阴男、阳女：年支对冲再逆数一位（如子对冲为午，逆数为巳）

更直观的查法——元辰为大耗星，与年支的关系如下：
- 子 → 未
- 丑 → 午
- 寅 → 巳
- 卯 → 辰
- 辰 → 卯
- 巳 → 寅
- 午 → 丑
- 未 → 子
- 申 → 亥
- 酉 → 戌
- 戌 → 酉
- 亥 → 申

口诀：**子未丑午寅巳卯辰，辰卯巳寅午丑未子，申亥酉戌亥申（元辰大耗）。**

**【属性】** 凶

**【含义与解读】**
元辰又名"大耗"，是命运中最凶的神煞之一，主破败、灾难、离散、大损。元辰入命者，一生中往往经历大的挫折——破产、离婚、重病、亲人离散等。

- **年柱见**：祖业败落，或家族有大变故。
- **月柱见**：青年时期有重大挫败。
- **日柱见**：自身一生有大起大落，婚姻易有重大变故。
- **时柱见**：晚年不得安宁，有重大损失。

**组合解读：**
- 元辰 + 七杀/羊刃：大凶之极，有生死之灾。
- 元辰 + 劫煞：破财达到极致，倾家荡产。
- 元辰 + 天乙贵人/天月二德：贵人救应，虽有大难却能保全性命。
- 元辰逢冲：灾难加倍，最凶组合之一。

**应用场景：** 判断一个人一生中有无重大灾难。元辰重者，必须谨慎规划人生，不要冒太大风险。

### 31. 亡神

**【查法】**
以年支为主，见四柱地支（三合局之帝旺位）：
- 寅午戌年 → 巳
- 巳酉丑年 → 申
- 申子辰年 → 亥
- 亥卯未年 → 寅

口诀：**寅午戌见巳，巳酉丑见申，申子辰见亥，亥卯未见寅。**

**【属性】** 凶

**【含义与解读】**
亡神主亡失、损失、精神不安、谋事难成。亡神入命者，常感情绪低落、精神压抑，做事容易半途而废或功败垂成。亡神又为"官符星"，与官司诉讼有关。

- **年柱见**：家族有破落之象，或长辈有牢狱之灾。
- **月柱见**：青年时期多挫折，精神压力大。
- **日柱见**：自身常有力不从心之感，功亏一篑之事多。
- **时柱见**：晚年不安，或子女惹上官司。

**组合解读：**
- 亡神 + 七杀：亡神带杀，官非牢狱之灾最重。
- 亡神 + 劫煞：双煞夹击，破财又伤身。
- 亡神 + 文昌：因文字、言论惹祸，文人被文字狱所害。
- 亡神 + 天乙贵人：有救，虽然波折多但最终能化险为夷。
- 亡神 + 空亡：凶性减弱，亡神落空反无所害。

**应用场景：** 看一个人的精神压力、事业挫折感，以及官非诉讼风险。

### 32. 孤辰寡宿

**【查法】**
以年支为主，见四柱地支：
**孤辰**（三合局前一位的空位）：
- 寅午戌年 → 寅前一位... 实际上：寅午戌见巳为孤辰
  正确查法：
  - 寅午戌年 → 巳（南）
  - 巳酉丑年 → 申（西偏南）
  - 申子辰年 → 亥（北偏西）
  - 亥卯未年 → 寅（东偏北）

**寡宿**（三合局后一位的空位）：
  - 寅午戌年 → 丑（北偏东）
  - 巳酉丑年 → 辰（东偏南）
  - 申子辰年 → 未（南偏西）
  - 亥卯未年 → 戌（西偏北）

更准确的：

**孤辰：**
- 寅卯辰年 → 巳
- 巳午未年 → 申
- 申酉戌年 → 亥
- 亥子丑年 → 寅

**寡宿：**
- 寅卯辰年 → 丑
- 巳午未年 → 辰
- 申酉戌年 → 未
- 亥子丑年 → 戌

（注：以上以四季为分类，是最通行的查法。）

口诀：**寅卯辰孤巳寡丑，巳午未孤申寡辰，申酉戌孤亥寡未，亥子丑孤寅寡戌。**

**【属性】** 凶

**【含义与解读】**
孤辰主孤独、无依；寡宿主寡居、失伴。二者常并见，主婚姻不顺、六亲缘薄、独居之象。孤寡入命者，在人群中常有格格不入之感。

- **年柱见**：祖上单传，或家族人丁不旺。
- **月柱见**：兄弟姊妹少，或与父母缘分薄。
- **日柱见**：婚姻不顺，夫妻感情冷淡，或聚少离多。
- **时柱见**：晚年孤独，子女不在身边。

**组合解读：**
- 孤辰 + 寡宿（双全）：一生孤独，婚姻难成或难守。
- 孤寡 + 华盖：清冷孤高，适合出家修行。
- 孤寡 + 驿马：孤身走天涯，漂泊无依。
- 孤寡 + 桃花：虽有异性缘但终归孤独，情路坎坷。
- 孤寡逢合（如巳申合）：孤独感被缓解，有人陪伴。

**应用场景：** 看婚姻是否孤独、六亲缘分薄厚，以及一个人是否适合独居或修行。

### 33. 十恶大败

**【查法】**
日柱为以下十日之一：
- 甲辰
- 乙巳
- 丙申
- 丁亥
- 戊戌
- 己丑
- 庚辰
- 辛巳
- 壬申
- 癸亥

口诀：**甲辰乙巳丙申丁亥，戊戌己丑庚辰辛巳，壬申癸亥十恶大败。**

**【属性】** 凶

**【含义与解读】**
十恶大败主破财、败家、不守祖业、挥霍无度。十恶大败日生者，对金钱的管理能力极差，有多少钱都能花光，且不善于理财守业。

- **年柱十恶大败**：祖上家业被败光，出身不佳。
- **月柱十恶大败**：青年时期大手大脚，难有积蓄。
- **日柱十恶大败（最重要）**：自身一生难聚财，赚多少花多少。
- **时柱十恶大败**：晚年贫穷，或因子女耗尽家财。

**组合解读：**
- 十恶大败 + 财星旺：赚得多花得更多，过路财神。
- 十恶大败 + 禄神：有工作有收入但存不住钱。
- 十恶大败 + 羊刃/劫煞：破财变本加厉。
- 十恶大败 + 天德/月德：有善行可弥补，虽能花钱但不至于败光。
- 十恶大败 + 空亡：空亡能削弱十恶大败的凶性，反而不太严重。

**实用建议：** 十恶大败日出生者，必须强制储蓄，如买房、买保险等硬性资产锁住资金，不可存现金。

**应用场景：** 判断一个人能否守住钱财、是否善于理财。对投资风险评估极有参考价值。

### 34. 阴差阳错

**【查法】**
日柱为以下十二日之一：
- 丙子
- 丁丑
- 戊寅
- 辛卯
- 壬辰
- 癸巳
- 丙午
- 丁未
- 戊申

- 壬戌
- 癸亥

口诀：**丙子丁丑戊寅辛卯，壬辰癸巳丙午丁未，戊申辛酉壬戌癸亥——阴差阳错日。**

**【属性】** 凶（主婚姻不顺）

**【含义与解读】**
阴差阳错是婚姻中最凶的神煞之一，主婚姻不顺、感情错位、姻缘差误。阴差阳错日出生者，婚姻中有"错位感"——不是在错误的时间遇到对的人，就是明明相爱却不能在一起。

**各日柱特质（差异）：**
- **丙子/丙午**：感情热烈但容易错失良缘。
- **丁丑/丁未**：婚姻中有委屈感，付出不被理解。
- **戊寅/戊申**：婚姻中界限感模糊，易有第三者问题。
- **辛卯/辛酉**：感情细腻敏感，容易因小事闹僵。
- **壬辰/壬戌**：感情中拖泥带水，当断不断。
- **癸巳/癸亥**：婚姻中压抑感重，沟通不畅。

**组合解读：**
- 阴差阳错 + 桃花：感情复杂，容易有多段感情或婚外情。
- 阴差阳错 + 孤辰寡宿：离婚或长期分居之象。
- 阴差阳错 + 天乙贵人：虽有贵人调解，但婚姻中的根本矛盾难消。
- 阴差阳错逢合：感情纠葛深，难以脱身。

**实用判断：**
- 男命日坐阴差阳错：容易娶二婚之妻，或对妻子不满意
- 女命日坐阴差阳错：所嫁非人，婚后悔意多

**应用场景：** 看婚姻质量、是否容易晚婚或离婚、婚姻中是否有根本性矛盾。

### 35. 红艳

**【查法】**
以日干为主，见四柱地支：
- 甲 → 午
- 乙 → 午
- 丙 → 寅
- 丁 → 未
- 戊 → 辰
- 己 → 辰
- 庚 → 酉
- 辛 → 酉
- 壬 → 子
- 癸 → 申

口诀：**甲午乙午丙寅丁未，戊辰己辰庚酉辛酉，壬子癸申红艳煞。**

**【属性】** 中性偏凶（主情色）

**【含义与解读】**
红艳主妖艳、风流、多情、魅力强。红艳入命者，男女长相有吸引力，气质独特，但感情上容易多情不专。红艳与桃花不同——桃花人人有，红艳则是"艳丽"加成。

- **年柱见**：少年早熟，有异性缘但感情不专。
- **月柱见**：青春时期感情丰富多变。
- **日柱见**：自身魅力强，但婚姻中易有外情。
- **时柱见**：晚年仍风姿绰约，一生桃花不减。

**组合解读：**
- 红艳 + 桃花：异性缘极旺，但也极容易因感情惹麻烦。
- 红艳 + 羊刃：情欲强烈，用情猛烈，爱恨分明。
- 红艳 + 天乙贵人：贵人多为异性，有借感情上位之象。
- 红艳 + 德秀贵人：有才情有魅力，风流倜傥但不下流。

**应用场景：** 判断一个人是否有特别的异性魅力，以及在感情中的忠诚度风险。

### 36. 天罗地网

**【查法】**
以年支/日支为主：
- **天罗**：辰（天罗为辰，即"辰为天罗"）
  - 更准确：戌亥为天罗（西北方）
- **地网**：巳（地网为巳，即"巳为地网"）
  - 更准确：辰巳为地网（东南方）

古籍主流查法：
- 年支见辰巳为地网，见戌亥为天罗
- 或者以纳音五行：火命人见戌亥为天罗，水土命人见辰巳为地网

最通用的查法：
- **辰、巳** 为地网
- **戌、亥** 为天罗

口诀：**辰巳为地网，戌亥为天罗。**

**【属性】** 凶

**【含义与解读】**
天罗地网主困顿、束缚、疾病、牢狱。落入天罗地网者，如入网之鱼，难以挣脱。命犯天罗地网之人，一生中常有受困之感——经济困顿、事业受制、病痛缠身或牢狱之灾。

- **天罗（戌亥）**：偏重精神上的困顿——压抑、抑郁、有志难伸。戌为火库，火命人最怕见戌亥。
- **地网（辰巳）**：偏重物质上的困顿——贫穷、负债、被限制自由。辰为水库，水/土命人最怕见辰巳。

**组合解读：**
- 天罗地网 + 七杀/羊刃：最凶，有牢狱之灾或重大疾病。
- 天罗地网 + 劫煞/亡神：雪上加霜，灾难叠加。
- 天罗地网 + 天月二德：凶中有救，虽被困但终能脱身。
- 天罗地网逢冲破：如辰戌冲、巳亥冲——反而破网而出，凶中转吉。

**应用场景：** 看人生中是否有"被困"的时期——经济困境、健康困境、或法律困境。

口诀：**子丧寅丑丧卯寅丧辰，卯丧巳辰丧午巳丧未，午丧申未丧酉申丧戌，酉丧亥戌丧子亥丧丑。**

**【属性】** 凶

**【含义与解读】**
丧门主丧事、哀事、亲人去世、悲伤之事。丧门入命或流年逢丧门，往往有亲人离世或参加葬礼之事。

- **年柱见**：祖上或家族中有早逝之人。

**组合解读：**
- 丧门 + 吊客：双丧组合，必有丧事，且不止一桩。
- 丧门 + 孤辰寡宿：送走亲人后更加孤独。
- 丧门 + 天月二德：虽遇丧事能妥善处理，不致太过悲伤难愈。

**应用场景：** 看人生中是否有重要的丧亲经历，以及某一年是否有丧事风险。

**【属性】** 凶

**【含义与解读】**
病符主疾病、身体不佳、精神状态差。病符入命或行运逢之，容易生病，身体抵抗力下降。虽不及灾煞凶险，但为长期的小病小痛。

- **年柱见**：体质偏弱，先天禀赋不足。
- **月柱见**：青少年时期多病。
- **日柱见**：中年体弱，小病不断。
- **时柱见**：老年多病，需注意保养。

- 病符 + 天月二德：病中有人照顾，能得康复。
- 病符 + 死/绝：久病难愈，缠绵病榻。
- 病符逢冲：突发疾病或旧疾加重。

**应用场景：** 看一个人身体健康程度和疾病倾向，以及流年是否有健康风险。

### 40. 官符（五鬼）

**【查法】**
以年支为主，见四柱地支：
- 子 → 辰
- 丑 → 巳
- 寅 → 午
- 卯 → 未
- 辰 → 申
- 巳 → 酉
- 午 → 戌
- 未 → 亥
- 申 → 子
- 酉 → 丑
- 戌 → 寅
- 亥 → 卯

官符实际上就是年支三合局之正对位（即年支顺数五位）。

口诀：**子辰丑巳寅午卯未，辰申巳酉午戌未亥，申子酉丑戌寅亥卯——官符五鬼名。**

**【属性】** 凶

**【含义与解读】**
官符又名"五鬼"，主官非、口舌、小人暗算、无端是非。官符入命者，容易莫名其妙惹上官司，或被人背后陷害。五鬼则更偏重小人作祟、阴私之事。

- **年柱见**：祖上有官非之灾。
- **月柱见**：青年时期容易与人发生口舌争执。
- **日柱见**：自身常被小人所扰，是非多。
- **时柱见**：晚年仍有人来找麻烦。

**组合解读：**
- 官符 + 勾绞：官非缠身，官司纠纷不断。
- 官符 + 亡神：牢狱之灾风险高。
- 官符 + 天乙贵人：有贵人帮忙化解是非。
- 官符 + 空亡：是非虽多但不至于酿成大祸。

**应用场景：** 看一个人是否容易招惹官非口舌、被小人算计。

## 五、附：神煞综合应用指南

### 5.1 神煞的轻重原则

不是见到吉神就好、见到凶煞就坏。神煞的应验程度取决于以下因素：

1. **神煞的旺衰**：得月令生旺者力量强，逢休囚死绝者力量弱
2. **神煞的数量**：同一神煞出现两次以上，力量倍增
3. **神煞的冲合**：被冲则破、被合则绊，都会影响神煞发挥
4. **五行生克**：神煞所临地支的五行生克关系
5. **日主强弱**：身强能任吉神，身弱则吉神反成压力

### 5.2 神煞的层次论

| 层次 | 神煞 | 影响力 |
|:---:|:---|:---:|
| 第一层（根本性） | 天乙贵人、天月二德、羊刃、魁罡 | 影响整个人生格局 |
| 第二层（阶段性） | 桃花、驿马、华盖、禄神 | 影响人生特定方面 |
| 第三层（事件性） | 天赦、天喜、丧门、病符 | 影响具体事件的发生 |

### 5.3 流年神煞应用

流年大运遇神煞，应验最快：
- **吉神流年**：天乙贵人、天德、月德、天喜——好事发生的年份
- **凶煞流年**：羊刃、劫煞、灾煞、丧门、吊客、病符——需要谨慎的年份
- **中性流年**：桃花、驿马——有变化但好坏看组合

### 5.4 常用神煞组合速查

| 组合 | 含义 |
|:---|:---|
| 天乙贵人 + 天月德 | 至贵至德，一生无大灾 |
| 羊刃 + 七杀 | 武职显贵，大将军命 |
| 桃花 + 羊刃 | 情伤血光，因色惹祸 |
| 驿马 + 禄神 | 动中得财，越忙越富 |
| 华盖 + 太极 | 玄学天才，适合学易 |
| 魁罡 + 天月德 | 以德制刚，大器晚成 |
| 亡神 + 七杀 | 官非牢狱，大凶之象 |
| 十恶大败 + 财星 | 过路财神，钱来钱去 |
| 阴差阳错 + 孤寡 | 婚姻不顺，孤独终老 |
| 天罗地网 + 空亡 | 破网而出，反为吉利 |
