## 一、三奇相关格局

### 1.1 奇仪合

| 项目 | 内容 |
|---|---|
| **名称** | 奇仪合 |
| **构成** | 天盘乙奇 + 地盘庚 (乙庚合)；天盘丙奇 + 地盘辛 (丙辛合)；天盘丁奇 + 地盘壬 (丁壬合) |
| **属性** | 吉 |
| **应用** | 合作、婚姻、谈判、联盟 |
| **注意** | 合中有情，但也可能牵绊。需配合门看：吉门则好事成，凶门则为事所累。 |

### 1.2 奇游禄位

| 项目 | 内容 |
|---|---|
| **名称** | 奇游禄位 |
| **构成** | 乙奇落震宫 (卯位，乙禄在卯)；丙奇落巽宫 (巳位，丙禄在巳)；丁奇落离宫 (午位，丁禄在午) |
| **属性** | 吉 |
| **应用** | 求职、加薪、求名、求禄 |
| **注意** | 禄位本身旺相有力，但需再查门是否被迫。门迫则禄位虽好却拿不到。 |

### 1.3 奇仪顺遂

| 项目 | 内容 |
|---|---|
| **名称** | 奇仪顺遂 |
| **构成** | 三奇在旺相之宫(月令相扶)，且不受门迫、不受刑墓 |
| **属性** | 吉 |
| **应用** | 一般性判断：整体局面顺畅 |
| **注意** | 此为综合断，非固定格局名。核心看"气是否流通"。 |

### 1.4 三奇入墓

| 项目 | 内容 |
|---|---|
| **名称** | 三奇入墓 |
| **构成** | 乙奇落乾六宫 (戌)；丙奇落乾六宫 (戌)；丁奇落艮八宫 (丑) |
| **属性** | 凶 |
| **应用** | 才华被埋没、有能力发挥不出、时机未到 |
| **注意** | 墓有"入"与"出"之分。月令冲开墓库或逢冲则解。如戌月入墓确实凶，辰月戌被冲则减凶。 |

### 1.5 三奇受制

| 项目 | 内容 |
|---|---|
| **名称** | 三奇受制 |
| **构成** | 乙奇落兑 (木被金克)；丙奇落坎 (火被水克)；丁奇落坎 (火被水克)；或三奇落死囚之宫 |
| **属性** | 凶 |
| **应用** | 好事难成、贵人被阻、能力受制 |
| **注意** | 若门吉可稍解，若门也凶则事不可为。 |

### 1.6 奇逢得使

| 项目 | 内容 |
|---|---|
| **名称** | 奇逢得使 / 三奇得使 |
| **构成** | 三奇临值使之门 (值使门所在宫见乙、丙、丁) |
| **属性** | 大吉 |
| **应用** | 办事有权、令出必行、领导支持 |
| **注意** | 值使门代表执行权，三奇临之则执行通顺。需值使门本身不被门迫。详见"三奇得使"节。 |

## 二、八门相关格局

### 2.1 门迫

| 项目 | 内容 |
|---|---|
| **名称** | 门迫 |
| **构成** | 门克宫：开门落震巽 (金克木)；休门落离 (水克火)；生门落坎 (土克水)；伤门落坤艮 (木克土)；杜门落坤艮 (木克土)；景门落乾兑 (火克金)；死门落坎 (土克水)；惊门落震巽 (金克木) |
| **属性** | 凶 |
| **应用** | 事被压制、条件不顺、强推必损 |
| **注意** | 门迫是奇门中最常用、最核心的凶断之一。门迫则门的力量发挥不出来甚至起反作用。 |

### 2.2 门制 (宫克门)

| 项目 | 内容 |
|---|---|
| **名称** | 门制 |
| **构成** | 宫克门：开门落离 (火克金)；休门落坤艮 (土克水)；生门落震巽 (木克土)；伤门落乾兑 (金克木)；杜门落乾兑 (金克木)；景门落坎 (水克火)；死门落震巽 (木克土)；惊门落离 (火克金) |
| **属性** | 凶 (比门迫稍轻) |
| **应用** | 阻力大、推进难、外部条件不配合 |
| **注意** | 宫克门为外势压内，比门迫(门克宫、内自损)轻一档。门制时人虽努力但有外部障碍。 |

### 2.3 门和 (门生宫)

| 项目 | 内容 |
|---|---|
| **名称** | 门和 |
| **构成** | 门生宫：开门落艮坤 (土生金)；休门落乾兑 (金生水)；生门落离 (火生土)；伤门落震巽 (木生木 = 比和，一般也算和)；杜门落震巽 (同伤门)；景门落巽 (木生火)；死门落乾兑 (金生土)；惊门落艮坤 (土生金) |
| **属性** | 吉 |
| **应用** | 事情有增量、顺遂、相得益彰 |
| **注意** | 比和亦算顺，但无增量，只是平顺。 |

### 2.4 门义 (宫生门)

| 项目 | 内容 |
|---|---|
| **名称** | 门义 |
| **构成** | 宫生门：开门落乾兑 (金见金=比和，也归为义)；休门落震巽 (水生木)；生门落坤艮 (土见土=比和)；景门落离 (火见火=比和)；死门落坤艮 (土见土=比和)；惊门落乾兑 (金见金=比和) |
| **属性** | 吉 (略次于门和) |
| **应用** | 外部支持、环境有利、贵人相助 |
| **注意** | 门义是"被外界照顾"之象，虽有助但自身未必有力。 |

### 2.5 门伏吟

| 项目 | 内容 |
|---|---|
| **名称** | 门伏吟 |
| **构成** | 值使门落本宫，天地盘同。整个盘中八门与原宫位相同 |
| **属性** | 凶 (主迟、主静) |
| **应用** | 不宜主动出击、适合守成、等待、积蓄 |
| **注意** | 伏吟不一定是死路。占事结果慢但稳，占人则不动、保守。逢冲则解。 |

### 2.6 门反吟

| 项目 | 内容 |
|---|---|
| **名称** | 门反吟 |
| **构成** | 值使门落对冲宫。整个盘中八门与本宫对冲 |
| **属性** | 凶 (主反复、主动) |
| **应用** | 事必反复、朝令夕改、行动快但不稳 |
| **注意** | 反吟不一定全部凶。占出行走动反而有利，占事业则主变动、折腾。 |

### 2.7 青龙返首 (戊+丙)

| 项目 | 内容 |
|---|---|
| **名称** | 青龙返首 |
| **构成** | 天盘戊 + 地盘丙 (天甲子戊 + 地丙奇) |
| **属性** | 大吉 |
| **应用** | 求财得财、谋事大利、官运亨通、出行顺利 |
| **注意** | 此为天盘戊与地盘丙的组合，非乙丙丁三奇的"返首"。戊+丙为"丙火生戊土"，戊为钱财资本，丙为火，主动能、贵人。若门吉则百事吉；若门迫则减吉。|

### 2.8 飞鸟跌穴 (丙+戊)

| 项目 | 内容 |
|---|---|
| **名称** | 飞鸟跌穴 |
| **构成** | 天盘丙 + 地盘戊 |
| **属性** | 大吉 |
| **应用** | 求财、求职、办事、出行皆吉 |
| **注意** | 丙+戊与戊+丙吉性相近，但丙主动，更利于"走上去"的主动作为。丙火入戊土之位，火土相生。 |

### 2.9 青龙逃走 (乙+辛)

### 2.10 白虎猖狂 (辛+乙)

| **注意** | 辛+乙为"虎欺龙"，主强势方欺凌弱势方。占婚姻主男方暴虐、家暴倾向。占事业主领导层冲突。 |

### 2.11 朱雀投江 (丁+癸)

## 三、九星相关格局

### 3.1 天辅时

| 项目 | 内容 |
|---|---|
| **名称** | 天辅时 |
| **构成** | 甲己日甲子时、乙庚日甲戌时、丙辛日甲申时、丁壬日甲午时、戊癸日甲辰时 (即六甲旬首时) |
| **属性** | 大吉 |
| **应用** | 百事皆宜。起局之时若逢天辅时，则万事可行 |
| **注意** | 天辅时是奇门最上吉的时格之一，此"天辅"非天辅星，而是"上天辅助"之意。六甲旬首之时，值符统领全局，格局纯正。 |

### 3.2 天网四张

| 项目 | 内容 |
|---|---|
| **名称** | 天网四张 |
| **构成** | 天盘癸 + 地盘癸、或时干为癸 |
| **属性** | 大凶 |
| **应用** | 被困、走投无路、网罗缠身、灾祸连绵 |

## 八、格局综合判断原则

### 8.1 格局层级

```
最优先：五不遇时 > 天网四张 > 龙返首/鸟跌穴 > 其他
```

- **时格**（五不遇时、天网四张） > **宫格**（门迫、击刑） > **名格**（具体格局名）
- 时格管整个盘的"气运"，宫格管局部，名格管细节

### 8.2 吉凶权重

| 层级 | 权重 | 说明 |
|---|---|---|
| 时干日干关系 | 最高 | 五不遇时直接判断此盘可用不可用 |
| 门迫/门制 | 高 | 八成以上的问题出在门上 |
| 十干克应 | 中高 | 天盘地盘组合决定事态性质 |
| 星神组合 | 中 | 九星定气运深浅，八神定事态性质 |
| 格局名称 | 低 | 有名格局不一定就是决定性因素 |

### 8.3 吉凶相抵原则

- 大吉格(龙返首/鸟跌穴等) + 门迫 = **吉中有阻，可做但费劲**
- 大凶格(白虎猖狂/朱雀投江等) + 生门/开门 = **凶中有救，谨慎可成**
- 五不遇时 + 任何吉格 = **仍以五不遇时为重，不建议行动**
- 天辅时 + 门迫 = **整体气运好，但具体操作不顺**
- 多个凶格聚集 = **事情不宜做，建议改期**

### 8.4 实战判断口诀

1. **先看时干克不克日干** — 克则改时
2. **再看用神落宫门迫否** — 迫则有阻力
3. **次看用神天地盘克应** — 定事态吉凶性质
4. **后看星神助还是压** — 定环境与人的因素
5. **格局名最后用来解释"象"** — 辅助描述而非下结论
