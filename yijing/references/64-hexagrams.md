# 六十四卦完整速查表

> 按《周易》卦序排列（乾1 → 未济64）
> 核心来源：《周易译注》（黄寿祺）、《周易正义》（王弼、孔颖达）、《易传》

---

## 卦序规律说明

六十四卦的排列并非随机，而是遵循**非覆即变**原则：

- **错卦（旁通）**：阴阳爻全部相反的卦，如乾⇄坤、泰⇄否。代表事物的对立面。
- **综卦（反对）**：将卦倒转180°所得的卦，如屯⇄蒙、需⇄讼。代表从不同角度观察同一事物。
- **互卦（互体）**：取一卦的二至五爻，下三爻为下互卦、上三爻为上互卦，揭示事物内部结构。
- **流行关系**：卦序体现"由简入繁、由治而乱、由始而终"的辩证逻辑。

## 上经（1–30）

### 1. ䷀ 乾（qián）
| 项目 | 内容 |
|------|------|
| **卦象** | ☰ 上乾 + ☰ 下乾 = 纯乾 |
| **卦辞核心** | 元亨利贞。四德俱全，创始、通达、和谐、正固。 |
| **彖传要义** | 天行健，乾道变化，各正性命；万物资始，统天。 |
| **象传要点** | 天行健，君子以自强不息。 |
| **现代应用** | 事业开创期，宜主动进取、坚持原则。领导力、创业、大格局之事。 |

| **现代应用** | 小有过失。宜低调、稳妥，不宜做大动作。谨慎行事、微调优化、保守策略。 |

### 63. ䷾ 既济（jì jì）
| 项目 | 内容 |
|------|------|
| **卦象** | ☵ 上坎（水）+ ☹ 下离（火）= 水火既济 |
| **卦辞核心** | 亨小，利贞。初吉终乱。 |
| **彖传要义** | 既济，亨小者，亨也。利贞，刚柔正而位当也。终止则乱，其道穷也。 |
| **象传要点** | 水在火上，既济。君子以思患而豫防之。 |
| **现代应用** | 已经成功、万事已济。创业成功但已见衰象。思患预防、居安思危。事情完成后的维护。 |

### 64. ䷿ 未济（wèi jì）
| 项目 | 内容 |
|------|------|
| **卦象** | ☹ 上离（火）+ ☵ 下坎（水）= 火水未济 |
| **卦辞核心** | 亨。小狐汔济，濡其尾，无攸利。 |
| **彖传要义** | 未济，亨。柔得中也。小狐汔济，未出中也。濡其尾，无攸利，不续终也。 |
| **象传要点** | 火在水上，未济。君子以慎辨物居方。 |
| **现代应用** | 尚未成功、事未成。虽未完成但充满希望。保持谨慎、继续努力。新的开始、未完待续。 |

## 错卦与综卦关系速查

| 本卦 | 错卦（旁通） | 综卦（反对） | 说明 |
|------|-------------|-------------|------|
| 乾䷀ | 坤䷁ | 乾䷀ | 乾综不变（上下皆阳） |
| 坤䷁ | 乾䷀ | 坤䷁ | 坤综不变（上下皆阴） |
| 屯䷂ | 鼎䷱ | 蒙䷃ | 屯倒转180°→蒙 |
| 蒙䷃ | 革䷰ | 屯䷂ | 蒙倒转→屯 |
| 需䷄ | 晋䷢ | 讼䷅ | 需倒转→讼 |
| 讼䷅ | 明夷䷣ | 需䷄ | 讼倒转→需 |
| 泰䷊ | 否䷋ | 泰䷊ | 泰综为泰（倒转不变） |
| 否䷋ | 泰䷊ | 否䷋ | 否综为否（倒转不变） |
| 既济䷾ | 未济䷿ | 未济䷿ | 既济倒转→未济 |
| 未济䷿ | 既济䷾ | 既济䷾ | 未济倒转→既济 |

> **规律**：凡六爻对称的卦（如泰否、既济未济），综卦与其自身或互成对方；
> 其余卦的综卦通常是相邻的一对（屯⇄蒙、需⇄讼、师⇄比等）。
> 错卦关系则贯穿全部，任何卦都有唯一对应的错卦。
