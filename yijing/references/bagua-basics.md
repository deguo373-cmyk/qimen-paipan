### 1.2 八卦三爻基本含义

- **初爻（下）**：代表根基、下层、开始、民众
- **中爻（中）**：代表核心、中坚、内在、人力
- **上爻（上）**：代表外在、结果、高层、天命

### 1.3 八卦取象口诀

> **乾三连**，坤六断；**震仰盂**，艮覆碗；
> **离中虚**，坎中满；**兑上缺**，巽下断。

## 二、先天八卦与后天八卦

### 2.1 先天八卦（伏羲八卦）

**卦序**：乾一、兑二、离三、震四、巽五、坎六、艮七、坤八

**方位**：
```
        乾
        ☰
        南
    ┌─────────┐
巽  │         │  兑
☴  │         │  ☱
东南│         │ 东南
    │    ☰    │
    │  南乾   │
    ├─────────┤
    │    ☷    │
    │  北坤   │
    │         │
艮  │         │  震
☶  │         │  ☳
东北│         │  东
    └─────────┘
        坎
        ☵
        北
```

> 注意：上南下北、左东右西，与地图相反。先天八卦图是"天地定位，山泽通气，雷风相薄，水火不相射"。

**应用场景**：
- 风水理气（玄空飞星等）
- 卦序数理（1-8的洛书数关联）
- 本体论：描述宇宙的"本来面目"、静态结构
- 卦象派生：六十四卦的先天卦序基础

### 2.2 后天八卦（文王八卦）

**卦序**：坎一、坤二、震三、巽四、中五、乾六、兑七、艮八、离九

**方位**：
```
        离
        ☲
        南
    ┌─────────┐
巽  │         │  坤
☴  │         │  ☷
东南│         │ 西南
    │    ☲    │
    │  南离   │
    ├─────────┤
    │    ☵    │
    │  北坎   │
    │         │
艮  │         │  乾
☶  │         │  ☰
东北│         │ 西北
    └─────────┘
        坎
        ☵
        北
```

**应用场景**：
- 奇门遁甲（九宫排盘基础）
- 八字风水（八卦方位对应宅位）
- 紫微斗数（部分宫位对应参考）
- 日常生活：描述事物的"当下状态"、动态变化
- 季节对应：震春、离夏、兑秋、坎冬

### 2.3 先天与后天核心区别

| 维度 | 先天八卦 | 后天八卦 |
|------|---------|---------|
| 创始人 | 伏羲 | 文王（据传） |
| 本质 | 体（本体、结构） | 用（功能、应用） |
| 描述 | 宇宙的本来秩序 | 万物的运行规律 |
| 方位基础 | 阴阳对待 | 五行顺布 |
| 卦序数 | 乾1→坤8 | 坎1→离9 |
| 季节 | - | 四季分明 |
| 风水应用 | 理气（玄空） | 形家、八宅 |
| 奇门应用 | 值符值使依据 | 九宫排盘基础 |

> **核心原则**："先天为体，后天为用"——先天讲的是宇宙结构与本质，后天讲的是时间变化与实际应用。

## 四、八卦与五行生克

### 4.1 五行对应

| 五行 | 卦 | 特性 |
|:----:|:--:|:----:|
| 金 | 乾、兑 | 刚健、果断、肃杀 |
| 木 | 震、巽 | 生长、发散、柔和 |
| 水 | 坎 | 流动、智慧、险陷 |
| 火 | 离 | 明丽、热情、急躁 |
| 土 | 坤、艮 | 中和、承载、静止 |

### 4.2 八卦生克关系（简要）

- **相生**：乾兑生坎（金生水）→ 坎生震巽（水生木）→ 震巽生离（木生火）→ 离生坤艮（火生土）→ 坤艮生乾兑（土生金）
- **相克**：乾兑克震巽（金克木）→ 震巽克坤艮（木克土）→ 坤艮克坎（土克水）→ 坎克离（水克火）→ 离克乾兑（火克金）
